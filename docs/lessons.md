# Mistakes & Lessons Learned

## RAG Mistakes

I underestimated how often retrieval quality, not model quality, was the root issue.

## Prompting Misconceptions

Long prompts did not consistently improve quality. Clear constraints and format expectations did.

## Overengineering Traps

Adding orchestration complexity too early slowed feedback loops and hid product priorities.

## What I'd Do Differently Now

Start lean, instrument from day one, and only add complexity after repeated evidence.
