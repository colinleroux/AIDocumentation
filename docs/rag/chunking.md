# Chunking

## Chunking Strategies

- Fixed-size chunks with overlap
- Structure-aware chunks (headings/sections)
- Hybrid chunking for mixed technical content

## Trade-offs (Size vs Context)

- Smaller chunks improve precision
- Larger chunks improve continuity
- Overlap helps recall but increases storage and noise

## What Worked for Me

Heading-aware chunking with moderate overlap has been the best default. I tune from retrieval misses, not guesswork.
